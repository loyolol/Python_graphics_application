# НКПиИТ - Система управления студентами и успеваемостью

Графическое приложение для автоматизации учета студентов, предметов и оценок в колледже. Разработано с использованием Python, Tkinter и MySQL.

## 🎨 Дизайн

Приложение выполнено в официальных цветах колледжа НКПиИТ:
- Темно-синий (#1D4287) — основной цвет
- Оранжевый (#FF7C20) — акцентный цвет для кнопок и подсветки

## 🚀 Функциональность

### Основные возможности:
- Управление студентами (добавление, редактирование, удаление, поиск)
- Управление предметами (добавление, редактирование, удаление)
- Учет успеваемости (выставление и редактирование оценок)
- Подключение к онлайн-базе данных MySQL

### Обязательные запросы:
- Создание таблиц — автоматически при первом запуске
- Добавление записей — во всех разделах
- Редактирование записей — во всех разделах
- Удаление записей — во всех разделах
